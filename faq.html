About File Objects

Does the "Subdivisions Surface" Add Modifier applied to the .blend file 3d models before exported to .3ds .dae .fbx .obj .stl .glb/.gltf?

  • Of course not, the exported files is a low-poly version and the Add Modifier that I always apply it is just "Mirror".

How did you get started design the 3d model?

Are the 3d models true size?

About Game Assets

Why not support the game ready (low poly)?

  • I don't have a background game industries, so if have it please share.
  • The 3D Assets available in quad, so you can easy to edit then modify to triangulate whatever you want.

Is the 3D Models from this site support for game assets?

  • I don't know it is support or not. I think you should to adjust a few things to make it work.

Why is the topology model so poor?

  • So many thanks for your feedback, I'm still learning to be 3D Modeler and share it with the sample render results here.

Why is the exported files from .Blend to other formats doesn’t include textures?

  • Most file formats just don't support exporting textures and only stores geometry data.
  • It does not keep anything regarding materials or textures. Therefore included HDRI and CC0 Texture source links reference from third parties

Why is the size of the .blend file so big, and the exported file size is so small?

  • As the description of the .blend file includes textures, hdri, and possibly materials. Whereas the export file for now is only an object.

About Texture/Materials & HDRI

Why don't some objects have texture images to download separately?

  • If it's from a third party, there I've included a link to download directly to the site.
  • Some are from materials (Procedural Textures in Blender), so I didn't include a link.
  • Sometimes I include in the .blend format object as a sample, if it may contain copyright or use permission that is not yet known.

Who do you contact to get permission from a third party?

  • Since this site is cc03ds.com (history here), I've contacted via email to Greg Zaal, Andreas Mischok, Sergej Majboroda from hdrihaven.com (now PolyHaven.com). And I've contacted via direct message on artstation to Lennart Demes from cc0textures.com (now ambientCG.com). They all gave me permission to post a direct link to the original file.
  • Outside of PolyHaven.com and ambientCG.com, I have no contact with anyone. So on the license I mention the point "this site just gives an sample for preview only". Then recommend to contact the respective owners.
